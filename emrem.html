EM

Em adalah unit yang digunakan untuk mengatur ukuran font atau element yang berdasarkan ukuran font dari element yang sekarang diterapkan. Ini memungkinkan Anda untuk mengubah ukuran font secara relatif terhadap element yang sekarang diterapkan.

REM

Rem (root em) adalah unit yang digunakan untuk mengatur ukuran font atau element yang berdasarkan ukuran font dari root element (elemen paling atas dari halaman web). Ini memungkinkan Anda untuk mengubah ukuran font secara global dengan mengubah ukuran font dari root element.
